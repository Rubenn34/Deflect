# Deflect

Deflect é um jogo desenvolvido utilizando **HTML, CSS e JavaScript**, criado como projeto de prática para explorar lógica de programação e interatividade no navegador.

---

## 🎮 Sobre o jogo

O objetivo do jogo é controlar o jogador e **desviar dos obstáculos**, acumulando pontos enquanto o jogo fica progressivamente mais desafiador.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🧠 Conceitos aplicados

* Manipulação do DOM
* Eventos de teclado
* Lógica de movimentação
* Sistema de pontuação
* Detecção de colisão
